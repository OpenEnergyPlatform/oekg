# Welcome to the OEKG Documentation

The OEP Factsheets are using the OEKG.

- S1: Name of study
- S2: Acronym
- S3: Institutions
- S4: Funding source
- S5: Research Question
- S6: Contact person
- S7: Sector divisions
- S8: Sectors
- S9: Energy carriers
- S10: Energy transformation processes
- S11: Descriptors (formally: Keywords)
- R1: Report Title
- R2: Date
- R3: DOI
- R4: Place of publication
- R5: Link
- R6: Authors

- SC7: Scenario types
